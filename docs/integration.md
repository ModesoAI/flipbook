# Integrating 3D Scroll Narrative into Your Project

This guide explains how to take the assets and logic generated by this pipeline and integrate them into an existing external website or application.

## 1. Assets (The Flipbook Frames)
The core of the experience is the optimized WebP frame sequence. 
- **Location:** `projects/{PROJECT_NAME}/www/assets/frames/`
- **Action:** Copy the entire `frames/` folder into your project's public assets directory.

## 2. Logic (The Flipbook Engine)
The rendering logic is encapsulated in a dependency-free ES6 module.
- **Location:** `projects/{PROJECT_NAME}/www/js/flipbook.js`
- **Action:** Copy this file to your JavaScript directory.

### Basic Integration Example:
```javascript
import { Flipbook } from './path/to/flipbook.js';

const scroller = new Flipbook({
    canvas: '#your-canvas-id',
    frameCount: [TOTAL_FRAMES], // Find this in index.html or project logs
    pathTemplate: './assets/frames/frame_${index}.webp'
});

scroller.init();
```

## 3. Styling
To achieve the 3D scroll effect, the canvas must be fixed to the viewport and the page must have enough height to scroll.
- **Location:** `projects/{PROJECT_NAME}/www/css/style.css`
- **Key styles to copy:**
    - `#scroll-canvas` (Fixed positioning, z-index: 1)
    - `.container` (Height: e.g., 600vh to define scroll speed)
    - `.scanlines` (Optional aesthetic overlay)

## 4. Performance Tips
- **Preloading:** The `Flipbook` class handles preloading automatically. Ensure you don't hide your loader until `onComplete` is called.
- **WebP:** Always use the WebP frames provided in the `www/` folder rather than the raw PNGs in `source/` for 60fps performance.
- **Aspect Ratio:** Ensure your canvas container matches the aspect ratio of the generated frames (Default is 16:9).
